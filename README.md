## :computer: Clone

##### 1 - Clonar o repositório
  `git clone https://github.com/me-wsantos/speech-recognition-windows.git`

##### 2 - Abrir a pasta do projeto e instalar as dependências
  `npm install`

##### 3 - Iniciar a aplicação
  `npm run dev`
